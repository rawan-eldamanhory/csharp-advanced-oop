# C# Advanced OOP Console Application

This C# console application demonstrates advanced Object-Oriented Programming (OOP) concepts.

## Concepts Demonstrated
- Encapsulation
- Constructors (default and parameterized)
- Method Overloading
- Inheritance
- Method Overriding
- Polymorphism
- Abstract Classes
- Interfaces
- Constants and Static Classes

## Classes
- Number1, Number2, Number3 – inheritance and overriding
- Number4, Number5 – abstract class and overrides
- Number7 – interface implementation
- MyConstant – static constant PI
